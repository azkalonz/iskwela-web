# Schoolhub_web

Web application

## Prerequisites

- `node`,`npm`

## Setup

- Clone the project

##### `git clone https://github.com/teamschoolhub/schoolhub_web.git`

- Enter the project directory

##### `cd schoolhub_web`

- Install dependencies

##### `npm install`

## Run

- Start react app from `schoolhub_web` #####`npm start`
